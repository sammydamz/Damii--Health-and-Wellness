# **App Name**: DAMII: Wellness Navigator

## Core Features:

- Wellness Input: Allows users to input descriptions of how they're feeling, covering both mental and physical well-being.
- AI-Powered Analysis and Support: Utilizes the Gemini API with Google Search grounding to analyze user input and provide empathetic, supportive responses. The AI acts as a 'tool', deciding what knowledge is helpful and relevant.
- Emotional Support Module: Delivers validation, empathy, and coping strategies for managing stress, anxiety, or low mood.
- Personalized Wellness Tips: Provides actionable, evidence-based suggestions on hydration, sleep hygiene, light exercises and healthy nutrition to address discomfort or low energy based on web searches.
- Activity and Mood Tracking: Enables users to log daily activities and moods, offering visual summaries (charts/graphs) to identify patterns and improve self-awareness. It will allow the user to select activies to correlate with wellness tip strategies

## Style Guidelines:

- Primary color: Soft lavender (#D1B0FF) to evoke calmness and relaxation.
- Background color: Light gray (#F5F5F5) for a clean, neutral backdrop that is easy on the eyes.
- Accent color: Muted teal (#73C2FB) for highlights and calls to action.
- Body font: 'Inter', a sans-serif for clean readability in the tips and descriptions.
- Headline font: 'Space Grotesk', a sans-serif that lends a futuristic, avant-garde flair, which contributes to the feeling of well-being and comfort with technology
- Simple, clean icons for navigation and to represent different wellness areas (e.g., sleep, nutrition, exercise).
- Clean, intuitive layout with clear sections for input, support, and wellness tips to minimize user stress.
- Subtle transitions and animations to create a smooth and engaging user experience.
